# Maximo AI Incident Investigator

## Overview

Maximo AI Incident Investigator is an AI-powered support copilot that helps IBM Maximo support teams troubleshoot incidents faster using Retrieval-Augmented Generation (RAG).

Instead of manually searching documentation, logs, XML payloads, and troubleshooting guides, users can ask natural language questions and receive evidence-backed answers.

---

## Problem Statement

Enterprise support engineers spend significant time investigating incidents by searching through logs, documentation, and historical tickets. This project aims to reduce troubleshooting time by providing AI-assisted root cause analysis and incident summaries.

---

## MVP Features

- Upload Maximo documentation
- Upload log files
- Upload XML payloads
- Ask questions in natural language
- Receive answers with citations
- Generate incident summaries
